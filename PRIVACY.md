# Privacy Policy - AI Assistant for Stream Deck

**Last Updated:** February 2025

## Data Collection

This plugin does **not** collect, store, or transmit any personal data to the plugin author or Elgato.

## Third-Party Services

When you use this plugin, your text content is sent to **your chosen AI provider's API**:

- **OpenAI** (api.openai.com)
- **Anthropic** (api.anthropic.com)
- **Google Gemini** (generativelanguage.googleapis.com)
- **OpenRouter** (openrouter.ai)
- **Custom endpoints** (user-configured URLs)

Your API key and text are transmitted directly from your computer to these services. Please review each provider's privacy policy:

- [OpenAI Privacy Policy](https://openai.com/privacy)
- [Anthropic Privacy Policy](https://www.anthropic.com/privacy)
- [Google Privacy Policy](https://policies.google.com/privacy)
- [OpenRouter Privacy Policy](https://openrouter.ai/privacy)

## Local Storage

The plugin stores the following settings **locally** on your device via Stream Deck's secure settings storage:

- API provider selection
- API key (encrypted by Stream Deck)
- Model selection
- Custom prompts
- User preferences (temperature, timeout, etc.)

## No Analytics

This plugin does not include any analytics, tracking, or telemetry.

## Data Retention

The plugin does not retain any processed text. All data is handled in-memory during processing and discarded immediately after.
